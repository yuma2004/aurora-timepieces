# AURORA TIMEPIECES
## 静寂の中に息づく時を超えた優雅さ

> *"時は川のように流れ、私たちは岸辺に立つ。しかし、真に価値あるものは、その流れの中で静かに輝き続ける星のような存在。"*

---

## 🕰️ プロジェクト概要

AURORA TIMEPIECESは、「静寂の中に息づく時を超えた優雅さ」をコンセプトとした高級時計ブランドのランディングページです。35-55歳の洗練された個人をターゲットに、瞑想的で上品な体験を提供します。

### 🎨 デザイン哲学

**コンセプト**: "Timeless Elegance in Serenity"
- **ムード**: 瞑想的、洗練された、静寂
- **美学**: 人間の手による温もり、ロボット的精密さの回避
- **レイアウト**: GASP原則（Generous, Architectural, Symphonic, Purposeful）

### テクスチャ & エフェクト
- **アナログ・グレイン**: 3-5%透明度のオーバーレイ
- **照明**: キアロスクーロ（明暗法）スタイル
- **アニメーション**: 控えめな優雅さ、250ms ホバーエフェクト
- **トランジション**: "wistful lull fade" 700ms

## 🚀 セットアップ手順

### 必要条件
- モダンなウェブブラウザ（Chrome 60+, Firefox 60+, Safari 12+）
- ローカルサーバー（開発用）

### インストール

1. **リポジトリのクローン**
   ```bash
   git clone https://github.com/yuma2004/aurora-timepieces.git
   cd aurora-timepieces
   ```

2. **ローカルサーバーの起動**
   ```bash
   # Python 3.x
   python -m http.server 8000
   
   # Python 2.x
   python -m SimpleHTTPServer 8000
   
   # Node.js (npx http-server)
   npx http-server -p 8000
   
   # PHP
   php -S localhost:8000
   ```

3. **ブラウザでアクセス**
   ```
   http://localhost:8000
   ```
